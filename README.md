# LinkSync Component

A premium, dark-aesthetic video chat application built with Vanilla JS and Vite.

## Features
- **Neon/Glassmorphism Design**: Modern UI with smooth animations.
- **Authentication**: Guest mode, Logic/Signup toggles, and persistent sessions.
- **Interest Matching**: Select from 16 categories before connecting.
- **Video Chat**: Full-screen immersive layout with mock WebRTC connections.
- **Social Tools**: Chat, Image sharing simulation, Friends list (mock).

## How to Run

1.  Open a terminal in this directory:
    ```bash
    cd linksync
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Start the development server:
    ```bash
    npm run dev
    ```

4.  Open the link shown in the terminal (usually `http://localhost:5173`) to view the app.

## Developer Notes
-   **Mock Data**: The app simulates peer connections using a placeholder video.
-   **Local Storage**: User sessions and preferences are saved in browser `localStorage`.
